## [0.1.0] - 2026-02-21

Initial release.

### Added

- Browse 50,000+ stations from the Radio Browser directory
- Discover popular and trending stations
- Search stations by name
- Save favorite stations with drag-to-reorder
- Add custom stations via stream URL
- Media key support (play/pause/stop from keyboard, headphones, Control Center)
- Now Playing info in macOS Control Center
- Volume control with persistence across launches
- Keyboard shortcuts (Cmd+P play/pause, Cmd+. stop, Cmd+F find, Cmd+1/2/3 navigate)
- Station favicons with disk cache
- About window with project info and support link
